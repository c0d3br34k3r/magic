# magic
A Java Library for Magic: The Gathering software

Currently in development, and any help appreciated!  Modeling Magic cards turned out to be much more complicated than I expected.

Dependencies:
* [Google Guava](https://github.com/google/guava)
* [Joda-Time](http://www.joda.org/joda-time)
* [JSR305](https://jcp.org/en/jsr/detail?id=305)
